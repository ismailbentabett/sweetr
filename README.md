
# Sweetr

twitter for people who cannot spell their Ts using solid js and firebaser and tailwind css



<img src="./public/logo.svg" width="250">


## Run Locally (Client)

```bash
  npm install
```

```bash
  npm run dev
```


## Run Locally (Server)

```bash
  cd api
```

```bash
  composer install
```

```bash
  php artisan migrate
```

```bash
  php artisan serve
```
